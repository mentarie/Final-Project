UI & UX

Apaitu UI dan UX?

UI Design adalah bagaimana suatu website atau aplikasi yang kamu buat terlihat seperti apa. Sedangkan UX Design itu proses membuat sebuah website atau aplikasi yang kamu buat menjadi mudah untuk digunakan dan tidak membingungkan ketika digunakan oleh pengguna.

Perbedaannya pada fokus utama. UI fokus pada kebutuhan pengguna terhadap program aplikasi aplikasi web atau mobile, sedangkan UX fokus pada pengalaman pengguna.
